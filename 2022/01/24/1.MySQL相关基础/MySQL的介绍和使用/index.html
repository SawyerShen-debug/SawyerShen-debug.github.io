MySql的介绍和使用
ShenXiaoYu 巴斯追光者
  2022-01-23 21:55:14 2022-01-23 21:55  
  • MySql基础 
 

一、MySQL的背景

  • 前身属于瑞典的一家公司，MySQL AB
  • 08年被sun公司收购
  • 09年sun被oracle公司收购

二、MySQL的优点

  1. 开源、免费、成本低
  2. 性能高、移植性也好
  3. 体积小，便于安装

三、MySQL的安装

属于C/S架构的软件，一般来讲安装服务端
请移步官网下载，配合workbench使用更佳

四、MySQL启用和停止

1.win启用和停止mysql

法一：计算机-> 右键管理-> 服务
法二：通过管理员身份运行
net start 服务名（启动服务）
net stop 服务名（停止服务）

2.mac启用和停止mysql

法一：系统偏好设置->mysql
法二：通过terminal->

  • 启动mysql：
    sudo /usr/local/mysql/support-files/mysql.server start
  • 停止mysql：
    sudo /usr/local/mysql/support-files/mysql.server stop

3.遇到的问题及解决办法

1.ERROR! MySQL server PID file could not be found!

手动关闭mysql服务在进行启动和停止就可以啦

2.zsh: command not found: mysql

解决方案：

  1. 在终端进入目录：/usr/local/bin
    cd /usr/local/bin/
  2. 在终端设置mysql命令路径
    sudo ln -fs /usr/local/mysql/bin/mysql mysql
  3. 输入mysql -u root -p

五、MySQL服务的登录和退出

法一：通过mysql自带的客户端
只限于root用户

法二：通过windows自带的客户端

  • 登录：
    • mysql 【 -h主机名 -P端口号】-u用户名 -p密码
  • 退出：
    • exit或ctrl+c

六、MySQL的常见命令

  1. 查看当前所有数据库
    show databases;
  2. 打开指定的库
    use 库名;
  3. 查看当前库的所有表
    show tables;
  4. 查看其他库的所有表
    show tables from 库名;
  5. 创建一个表
  6. 查看表结构
    desc 表名;
  7. 查看服务器版本
    法一：登录到mysql服务端
    select version()
    法二：没有登录服务端
  • win里
    mysql -v mysql -version

七、MySQL的语法规范

  1. 不区分大小写，但建议关键字大写，表名，列名小写
  2. 每条命令最好用分号结尾
  3. 每条命令根据需要，可以进行缩进或换行
  4. 注释
  • 单行注释：#注释文字
  • 单行注释：-- 注释文字（记得--后的空格）
  • 多行注释：/* 注释文字 */
